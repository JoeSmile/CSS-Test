Syntax:

@media not|only mediatype and (media feature and|or|not mediafeature) { CSS-Code; }

    media type

  • all: 默认值，所有设备
  • print: Used for printer(打印机)
  • screen: Used for computer screens, tables,smart-phones etc(电脑，平板，智能手机等)
  • speech: User for screenreaders
  • PS：一般我们需要用的，同时也是最常用的就是screen。

    media features

    feateure的值很多，比如color（The number of bits per color component for the output device）, grid（Whether the device is a grid or bitmap）等有关设备信息的值。这里我们介绍常用的几个。

  • height: the viewport height(视窗高度)。可加 min/max 前缀。
  • width: the viewport width(视窗宽度)。可加 min/max 前缀。(当width为800px时生效， min-width为屏幕宽度>= 800, max-witdh为屏幕宽度 <= 800) @media screen and (width: 800px) { ... }
  • orientation： The orientation（方向） of the viewport (landscape(横屏) or portrait（竖屏） mode) @media all and (orientation: portrait) { ... }
  • resolution： The resolution of the output device（屏幕分辨率), using dpi or dpcm
  • ratio: device-aspect-ratio（设备宽高比) @media screen and (device-aspect-ratio: 16/9), screen and (device-aspect-ratio: 16/10) { ... }

    Key words

  • and: 关键字用于合并多个媒体属性或合并媒体属性与媒体类型。
  • only:
  • not: 在媒体查询为假时返回真。not关键字仅能应用于整个查询，而不能单独应用于一个独立的查询。not的用法比较绕，建议用and。 @media not all and (monochrome) { ... }
