定义： 元素可提供有关页面的元信息（meta-information），比如针对搜索引擎和更新频度的描述和关键词。 理解： meta不显示在网页中，但可以被浏览器解析并控制浏览器的行为。
常用meta属性及说明： 1.keywords：用于告诉搜索引擎，你网页的关键字 eg: 2.description：用于告诉搜索引擎，你网站的主要内容 eg: 3.viewport：移动端的窗口行为 这个配置的意思：让viewport的宽度等于物理设备上的真实分辨率，不允许用户缩放。主流app一般都这么设置。 关于viewport： width：控制 viewport 的大小，可以指定的一个值，如果 600，或者特殊的值，如 device-width 为设备的宽度（单位为缩放为 100% 时的 CSS 的像素）。 height：和 width 相对应，指定高度。 initial-scale：初始缩放比例，也即是当页面第一次 load 的时候缩放比例。 maximum-scale：允许用户缩放到的最大比例。 minimum-scale：允许用户缩放到的最小比例。 user-scalable：用户是否可以手动缩放。 4.robots: 用来告诉爬虫哪些页面需要索引，哪些页面不需要索引 具体参数如下： none : 搜索引擎将忽略此网页，等价于noindex，nofollow。 noindex : 搜索引擎不索引此网页。 nofollow: 搜索引擎不继续通过此网页的链接索引搜索其它的网页。 all : 搜索引擎将索引此网页与继续通过此网页的链接索引，等价于index，follow。 index : 搜索引擎索引此网页。 follow : 搜索引擎继续通过此网页的链接索引搜索其它的网页 5.author: 用于标注网页作者 6.copyright:版权 7.renderer: 指定浏览器默认以何种方式渲染页面 //默认webkit内核 //默认IE兼容模式 //默认IE标准模式 8.字符集设置（title标签在浏览器显示乱码的bug，可以通过设置这个解决） 9. X-UA-Compatible：浏览器采取何种版本渲染当前页面 //指定IE和Chrome使用最新版本渲染当前页面 10.Set-Cookie： cookie设置 //格式 //具体范例
